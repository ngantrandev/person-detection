                    CHÚ Ý

Để cài đặt thư viện cần thiết: pip install -r requirements.txt


                    MÔ TẢ
Đường dẫn input đầu vào nằm ở đầu file main.py


Để chạy chương trình:

        python main.py

Nếu muốn chỉ định model, chế độ stream( lấy video local hay internet), input ... thì có thể thực hiện như sau

Ví dụ chọn model có đường dẫn mobidelnet_ssd/model.caffemodel thì thực hiện:
                            python main.py --model mobidelnet_ssd/model.caffemodel
